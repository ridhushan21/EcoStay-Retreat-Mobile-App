# 🌿 EcoStay Retreat Mobile App

EcoStay Retreat is a native Android mobile application designed to enhance the experience of eco-conscious travelers by providing a seamless platform for booking eco-friendly accommodations, reserving outdoor activities, and exploring sustainable tourism services.

## 🚀 Features

* 🔐 User Authentication (Login & Registration)
* 🏡 Eco-friendly Room Booking with Date Selection
* 🌄 Outdoor Activity Reservations (Hikes, Workshops, Tours)
* 📍 Explore Nearby Attractions & Sustainability Initiatives
* 🔔 Real-time Notifications (Bookings & Promotions)
* 👤 User Profile Management
* ✅ Form Validation & Secure Data Handling

## 💡 Project Overview

This application was developed to replace traditional booking methods with a modern digital solution that improves convenience, efficiency, and user engagement. The system allows users to manage bookings, explore services, and receive updates through a single mobile platform.

## 🛠️ Tech Stack

* **Language:** Java
* **Platform:** Android (Android Studio)
* **UI Design:** XML
* **Database:** SQLite
* **Tools:** Android Studio, GitHub

## 🧠 Key Functionalities

* 📅 Booking system with integrated date picker
* 💾 SQLite database for local data storage
* 🔄 SharedPreferences for session management
* 🔔 Android Notification API for alerts and updates
* 🧪 Input validation and error handling

## 📂 Project Structure

* `LoginActivity` – User authentication
* `RegisterActivity` – User registration
* `DashboardActivity` – Main navigation
* `RoomBookingActivity` – Room booking system
* `ActivityReservationActivity` – Activity booking
* `ProfileActivity` – User profile management
* `DatabaseHelper` – Database operations

## 🎯 Objective

The main goal of this project is to provide a user-friendly and efficient mobile solution that promotes sustainable tourism while improving booking experiences through digital transformation.
